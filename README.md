# Retail Data Pipeline 🚀

Este proyecto es un pipeline de ingeniería de datos desarrollado en **Python** para procesar información de ventas de retail.

## 🛠️ Tecnologías utilizadas
* **Lenguaje:** Python 3.x
* **Base de Datos:** PostgreSQL
* **Control de Versiones:** Git & GitHub

## 📋 Funcionalidades
1. **Ingesta de Datos:** Carga manual y desde archivos CSV.
2. **Transformación:** Limpieza de datos duplicados y manejo de nulos.
3. **Persistencia:** Almacenamiento estructurado en base de datos relacional.

## 🚀 Cómo ejecutarlo
1. Clonar el repositorio.
2. Configurar el entorno virtual: `python -m venv venv`.
3. Ejecutar los scripts en orden: `01_carga_manual.py`, `02_carga_csv.py`, etc.
